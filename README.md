# Colmar Academy

This project was a part of Codecademy's curriculum of building a Website with HTML, CSS, Bootstrap, and Github Pages Path. This project let me applied everything I had learned to build a responsive site from scratch. Responsive design techniques and media queries were applied to create desktop and mobile versions of the web page.

## Technologies

This project was created with:

- CSS3
- Bootstrap
